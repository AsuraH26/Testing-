INSTALLATION 

So wird das Spiel installiert 
Legen Sie die DVD mit S.T.A.L.K.E.R.: Call of Pripyat in Ihr DVD-ROM-Laufwerk. Wenn das Installationsprogramm nicht automatisch startet, sobald das Betriebssystem die DVD erkannt hat, können Sie hierzu manuell über den Explorer die Datei "Setup.exe" ausführen.

Wir empfehlen Ihnen alle Anti-Virenprogramme abzuschalten, da diese die Geschwindigkeit der Spielinstallation signifikant verringern.

MEHRSPIELERMODUS VON S.T.A.L.K.E.R.: Call of Pripyat

Verbindung mehrerer Spieler über LAN und Direct IP:

Die Spieler können sich miteinander über ein Local Area Network (LAN) oder per Direct IP über das Internet verbinden. Beide Verbindungstypen nutzen das TCP/IP-Protokoll. Um bei einem beliebigen Spieltyp der Host zu sein, müssen Sie im Hauptmenü die Option "Netzwerkspiel" auswählen.

Produktschlüssel benötigt
o Sie müssen S.T.A.L.K.E.R.: Call of Pripyat mit einem gültigen Produktschlüssel installiert haben, um online eine Verbindung herzustellen und zu spielen. Ein Produktschlüssel kann nicht von zwei verbundenen PCs gleichzeitig benutzt werden. Wenn Sie beim Aufbau der Verbindung also eine Fehlermeldung bezüglich Ihres Produktschlüssels erhalten, dann installieren Sie S.T.A.L.K.E.R.: Call of Pripyat bitte neu und geben Sie einen gültigen Produktschlüssel ein.

Firewall-Einstellungen
o Das Ausrichten von Spielen im Mehrspielermodus unter Verwendung von Router/Firewall/Proxy wird derzeit nicht unterstützt. Mit den folgenden Informationen können sich Spieler mit einem Host verbinden.
Befinden Sie sich hinter einer Firewall oder einem Proxy-Server und können Sie deren Einstellungen verändern, dann braucht S.T.A.L.K.E.R.: Call of Pripyat die folgenden offenen TCP-Ports, um funktionieren zu können:
6667 (IRC)
3783 (Voice Chat Port) 
27900 (Master Server UDP Heartbeat) 
28900 (Master Server List Request) 
29900 (GP Connection Manager) 
29901 (GP Search Manager) 
13139 (Custom UDP Pings) 
6515 (Dplay UDP)
6500 (Query Port)
Bitte achten Sie darauf, dass S.T.A.L.K.E.R.: Call of Pripyat in den Ausnahmelisten Ihrer Windows Firewall und anderer Firewalls aufgeführt ist, die auf Ihrem Rechner aktiv sind. Befinden Sie sich hinter einem DMZ, sollten Sie alle On-Access-Scanner deaktivieren.

Das Installationsprogramm von S.T.A.L.K.E.R. fügt das Spiel automatisch der Windows-Firewall-Ausnahmeliste hinzu.

Anti-Virus Software
o Viele Benutzer von Anti-Virus Software (wie Nvidia Access Manager, NOD32, Norton, PandaSoft etc.) sind nicht in der Lage, eine Verbindung zu GameSpy herzustellen, bemerken eine schlechte Rechnerleistung oder sogar Abstürze des Spiels. Wenn also Probleme auftreten, schlagen wir vor, dass Sie S.T.A.L.K.E.R.: Call of Pripyat auf die Ausnahmelisten dieser Programme setzen oder diese während des Spiels deaktivieren.

Router mit symmetrischem NAT
o Bestimmte Router mit symmetrischem NAT funktionieren nicht gut mit S.T.A.L.K.E.R.: Call of Pripyat. Diese Router sind sehr selten und für die meisten gibt es ein Firmware-Update, das das Problem behebt. Erkundigen Sie sich daher beim Hersteller des Routers nach aktueller Firmware.

TIPPS ZUR RECHNERLEISTUNG

Um die Rechnerleistung zu verbessern, können Sie im Hauptmenü die "Optionen" aufrufen und dann den Tab "Video" auswählen. Jede der folgenden Optionen sollte die Rechnerleistung verbessern können:

- Ändern Sie "Render" auf "Statische Beleuchtung"
- Ändern Sie die Qualitätseinstellung auf "Mittel", "Niedrig" oder "Minimum"
- Verringern Sie die Bildauflösung auf 1024x768 oder weniger
- Aktivieren Sie den Vollbildmodus
- Verringern Sie die Sichtweite, das anisotrope Filtern und die Schattenqualität

Hintergrundanwendungen
Programme, die im Hintergrund laufen, während man sich im Internet befindet (wie ICQ, Internet-Browser, VOIP-Anwendungen oder File Sharing Software) beeinflussen die Spielgeschwindigkeit von S.T.A.L.K.E.R.: Call of Pripyat. Das Schließen dieser Programme kann die Geschwindigkeit, mit der das Spiel läuft also erhöhen. Anti-Virus Programme und Firewalls bremsen den Rechner in aller Regel ebenfalls aus. Wenn Sie mit solchen Programmen Probleme haben, sollten Sie sie so einstellen, dass S.T.A.L.K.E.R.: Call of Pripyat von Sicherheitsüberprüfungen ausgenommen wird. Weitere Informationen hierzu erhalten Sie vom jeweiligen Hersteller der Software.

Arbeitsspeicher
Eine weitere Methode, die Leistung von S.T.A.L.K.E.R.: Call of Pripyat zu erhöhen, ist die Vergrößerung des Arbeitsspeichers. S.T.A.L.K.E.R.: Call of Pripyat kann große Mengen Speicher belegen, so dass das System eventuell gezwungen ist, virtuellen Speicher zu nutzen. Dieser Vorgang ist dann langsamer als die direkte Benutzung von RAM. Bei einfachen Systemen mit 512MB RAM kann eine Vergrößerung des Arbeitsspeichers also dazu führen, das Spiel zu beschleunigen. Bitte stellen Sie sicher, dass der virtuelle Arbeitsspeicher des Betriebssystems aktiviert ist.

Dual Core
S.T.A.L.K.E.R.: Call of Pripyat nutzt Ihren Dual und Quad Core Prozessor automatisch. Dies ist eine der besten Methoden, die Leistung zu verbessern.

Aktuelle Grafikkartentreiber
Wir empfehlen Ihnen dringend, die aktuellsten Treiber für Ihre Grafikkarte zu installieren. Diese können von der Website des Herstellers der Grafikkarte heruntergeladen werden. Bitte besuchen Sie dazu die Websites http://www.nvidia.com/ oder http://www.ati.com/, je nachdem, welche Grafikkarte Sie benutzen.

Aktuelle Soundkartentreiber
Wenn Sie mit der Soundkarte oder allgemeine Leistungsprobleme mit dem System haben, sollten Sie sich informieren, ob neue Treiber für Ihre Soundkarte verfügbar sind. Diese sind in aller Regel auf der entsprechenden Website zu finden.

Motherboard-Treiber
Beim Aktualisieren von Systemen werden Treiber von Motherboards oftmals übersehen. Eine Folge davon ist, dass viele Computer mit alten Versionen laufen und daher nicht richtig bzw. schnell funktionieren. Bitte sehen Sie auf der Website des Herstellers Ihres Motherboards nach, ob neue Treiber zur Verfügung stehen.

6.	Folgen Sie den Anweisungen auf dem Bildschirm.
